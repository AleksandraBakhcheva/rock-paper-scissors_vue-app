# "Rock Paper Scissors"

Rules: If the player wins, he receives 1 point. If the computer wins, the player loses 1 point. 

### Original Game      
- Paper beats Rock
- Rock beats Scissors
- Scissors beat Paper

## Stack

- Vue3
- Vuex
- Vue Router
- Vue Page Transition
- JavaScript
- HTML5
- Sass
- Gsap

## Start app

```bash
clone the project
cd "target folder"
npm install
npm run serve
```
